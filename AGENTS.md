# Instrucciones para Codex

Este proyecto debe desarrollarse con:

- React
- Vite
- JavaScript
- CSS modular o CSS normal, evitando librerías innecesarias
- Código claro, ordenado y fácil de mantener
- Componentes reutilizables
- Diseño responsive para móvil, tablet y escritorio

## Reglas

1. Antes de modificar archivos, explica brevemente qué vas a cambiar.
2. No borres código existente sin avisar.
3. No instales dependencias sin justificarlo.
4. Mantén los componentes pequeños.
5. Usa nombres claros para variables, funciones y componentes.
6. Evita soluciones complicadas si hay una forma sencilla.
7. Después de cada cambio importante, indica qué archivo se modificó.
8. Si hay errores, corrígelos con cambios mínimos.

## Estructura recomendada

src/
  components/
  pages/
  hooks/
  utils/
  styles/
  assets/

## Estilo visual

La interfaz debe ser moderna, limpia, responsive y fácil de usar.
